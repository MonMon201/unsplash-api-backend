## Description

[Unsplash](https://unsplash.com/) Api application. Backend part.

## Installation

```bash
$ npm i
```

## Set up
Copy environment variables and fill in a .env file

```bash
$ cp .env.example .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
